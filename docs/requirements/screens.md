# 画面一覧（v0.1・軽量版）

要件定義書（requirements.md）から、画面に相当するものを抽出した一覧。レイアウトの詳細は基本設計で決めるため、ここでは「どんな画面があり、何ができるか」までを軽くまとめる。

## 画面一覧表

| # | 画面名 | 役割 | できること |
|---|---|---|---|
| 1 | メモ一覧（フォルダビュー） | フォルダ階層をたどり、メモを探す起点となる画面 | フォルダを開く・作る・名前を変える・移動する、メモを選んで開く |
| 2 | メモ詳細 | 1つのメモを開いた状態。段落が簡略表示で並ぶ | 段落をタップしてモーダル表示／「＋」でインライン展開、段落編集モードに入る |
| 3 | 段落の全文表示（モーダル） | 段落をタップしたときに手前に重なって表示される画面 | 全文を読む、さらに内側の段落をタップしてさらに重ねる、右スワイプで戻る |
| 4 | 段落編集モード | 段落の構造を編集するための専用モード | 範囲選択・段落化・段落解除・タグ付け/解除・段落結合・段落分離 |
| 5 | 検索 | キーワード・タグで検索する入力画面 | 検索対象（メモ単位/段落単位）・検索条件（キーワード/タグ/組み合わせ）を指定する |
| 6 | 検索結果 | 検索結果を表示する画面 | 結果を一覧 or 結合表示で読む、「道順を確認」「ジャンプ」を選んで初めて移動する |
| 7 | タグ管理 | タグの色・名前を確認・編集する画面 | タグの色を変更する、配色パレット（標準/鮮やか）を切り替える |
| 8 | 設定 | アプリ全体の設定をまとめる画面 | マーカー記号の変更、配色パレットの選択、オンボーディングの再視聴、バックアップ設定（有料） |
| 9 | オンボーディング | 初回起動時の説明・初期設定画面 | 操作方法のデモを見る、初期設定（配色等）を選ぶ |
| 10 | 移動先選択 | フォルダ/メモの移動先となるフォルダを選ぶための画面。元の画面の手前にほぼ全画面で重なって表示される | フォルダ名をタップして階層を移動、＜戻る/右スワイプで1階層戻る、「ここに移動」で確定、「キャンセル」で元の画面に戻る |

## 簡易ワイヤーフレーム（主要3画面のみ）

### メモ一覧（フォルダビュー）
```
┌─────────────────────┐
│ ＜戻る   フォルダ名      ⚙設定 │
├─────────────────────┤
│ 📁 マットピラティス          │
│ 📁 マシンピラティス          │
│ 📄 今後やりたいこと          │
│ 📄 買い物リスト             │
├─────────────────────┤
│      ＋新規作成   🔍検索      │
└─────────────────────┘
```

### メモ詳細（段落が簡略表示で並ぶ）
```
┌─────────────────────┐
│ ＜戻る   メモ名      編集モード │
├─────────────────────┤
│ ⚪️エクササイズA          ＋│
│ ⚪️エクササイズB          ＋│
│ （ラベルなしの普通の文章...） │
└─────────────────────┘
```

### 検索結果
```
┌─────────────────────┐
│ ＜戻る   検索結果（3件）        │
├─────────────────────┤
│ 「腹筋」を含む段落            │
│ ──────────────── │
│ ・段落の中身がここに表示される   │
│   [道順を確認] [ジャンプ]    │
│ ──────────────── │
│ ・別の段落の中身              │
│   [道順を確認] [ジャンプ]    │
└─────────────────────┘
```

## 未確定事項

- 設定画面内の各項目（マーカー・配色・バックアップ等）を1画面にまとめるか、サブ画面に分けるかは設計フェーズで検討

## 決定事項（設計フェーズで確定）

- フォルダ・メモの名前変更・削除・移動は、長押しで表示されるメニュー（名前を変更／移動／削除）から行う。移動を選んだ場合は#10「移動先選択」画面に遷移する（詳細は`docs/basic-design/screens/memo-list.md`）
