# 画面設計書：新規作成（フォルダ/メモのインライン作成）

要件定義（`docs/requirements/screens.md`の画面一覧表#1）の「＋新規作成」操作に対応する設計書。独立した画面ではなく、メモ一覧（[memo-list.md](./memo-list.md)）上の状態変化として実現する。選択ボトムシートのレイアウトは[create-item-select.excalidraw](./create-item-select.excalidraw)を参照。

## 画面の役割
フォルダまたはメモを新規作成し、その場で名前をつける。

## 操作フロー

```
＋新規作成タップ
   ↓
選択ボトムシート表示（フォルダ / メモ / キャンセル）
   ↓ キャンセルをタップ
   → ボトムシートが閉じる（作成しない）

   ↓ フォルダまたはメモを選択
一覧の先頭に仮名付きアイテムが追加され、名前が編集状態になる
   ↓
名前を入力
   ↓
確定または外タップ
   → 名前あり：アイテムを保存
   → 名前が空欄：作成自体をキャンセルし、アイテムがリストから消える
```

## ステップ1：選択ボトムシートの項目一覧表

「＋新規作成」タップで画面下部からボトムシートが出現する。ボトムシートが出ている間、後ろのメモ一覧は少し暗くなる（5-2章「重なりの視認性」と同じルールを適用）。

| 項目名 | 種別 | イベント |
|---|---|---|
| 📁 フォルダを作成 | インプット（操作トリガー） | タップでボトムシートを閉じ、メモ一覧の先頭に「新しいフォルダ」という仮名のフォルダを追加して編集状態にする |
| 📄 メモを作成 | インプット（操作トリガー） | タップでボトムシートを閉じ、メモ一覧の先頭に「新しいメモ」という仮名のメモを追加して編集状態にする |
| キャンセル | インプット（操作トリガー） | タップでボトムシートを閉じる（何も作らない） |

```
選択ボトムシート表示中:
┌──────────────────────────────────────────┐
│ ＜ 仕事フォルダ                     ⚙   │  ← 暗くなる
├──────────────────────────────────────────┤
│  📁 プロジェクトA                       │  ← 暗くなる
│  📄 今日のタスク                        │  ← 暗くなる
│                                          │
│ ┌──────────────────────────────────────┐ │
│ │  📁 フォルダを作成                   │ │
│ ├──────────────────────────────────────┤ │
│ │  📄 メモを作成                       │ │
│ ├──────────────────────────────────────┤ │
│ │  キャンセル                          │ │
│ └──────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

## ステップ2：インライン編集状態

選択後、新しいアイテムが一覧の適切な位置に追加される。仮名はすべて選択された状態で表示され、そのまま入力すると仮名が上書きされる。キーボードが表示され、一覧は上にスクロールする。

追加される位置は種別によって異なる：

| 種別 | 編集中の表示位置 | 確定後の位置 |
|---|---|---|
| フォルダ | フォルダセクションの先頭 | 五十音順の正式な位置へ自動移動 |
| メモ | メモセクションの先頭 | そのまま（最終更新日が今なので先頭が正しい位置） |

フォルダ・メモそれぞれのセクション先頭に置くことで、編集中も「何を作っているか」の文脈が分かりやすくなる。メモを作成する場合、メモセクション先頭が画面外にある場合はそこまで自動スクロールする。

| 仮名の種別 | デフォルト文字列 |
|---|---|
| フォルダ | 新しいフォルダ |
| メモ | 新しいメモ |

```
インライン編集中（「新しいフォルダ」を作成した場合）:
┌──────────────────────────────────────────┐
│ ＜ 仕事フォルダ                     ⚙   │
├──────────────────────────────────────────┤
│ ┌──────────────────────────────────────┐ │
│ │  📁 [新しいフォルダ]（選択状態）      │ │  ← 編集中アイテム（先頭に固定）
│ └──────────────────────────────────────┘ │
│  📁 プロジェクトA                       │
│  📄 今日のタスク                        │
│                                          │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  [キーボードが表示される]                │
└──────────────────────────────────────────┘
```

## 確定/キャンセルの挙動

| 操作 | 名前の状態 | 結果 |
|---|---|---|
| キーボードの確定ボタン | 名前あり | 名前を確定しアイテムを保存。編集モード解除 |
| 編集対象以外をタップ | 名前あり | 同上 |
| キーボードの確定ボタン | 空欄 | 作成をキャンセル。アイテムがリストから消える |
| 編集対象以外をタップ | 空欄 | 同上 |
